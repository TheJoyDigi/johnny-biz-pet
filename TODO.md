# Blog posts schedule
## This is the guide line for the works for blog posts scheduler and tasks
### Mission
Your mission is to learn the current blog post machanism and plan blog post schedule for the next 3 months that best for SEO and traffic.

### Phase 1: Foundation
- [ ] Learn the current blog post machanism
- [ ] Write down the list of SEO keywords target in Markdown file docs/seo_keywords.md based on content of the website
- [ ] Based on the SEO keywords target, plan blog post schedule in Markdown file docs/blog_posts_schedule.md for the next 3 months, 2 post weekly

### Phase 2: Implementation
- [ ] create promt to go to task list docs/blog_posts_schedule.md and start to implement 1 task, update the task status to "Done" when finished, store promt in file docs/blog_posts_prompts.md

## Important notes
- Always use nanobanana for generate images for blog content
- Blog post is human like written, not AI generated
- Blog post much create **real value** for the reader, while helping the website to rank higher in search engine
- Blog post must be unique, not copy from other website
